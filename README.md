# Saúde Cardíaca - Monorepo

Este repositório contém o código-fonte do projeto Saúde Cardíaca, estruturado como um monorepo que contempla a API e a aplicação Web (cliente).

## Estrutura de Diretórios

- `/api`: Contém o backend desenvolvido em Spring Boot (Java).
- `/web`: Contém a aplicação frontend (cliente).

## Como Executar

Para executar cada um dos projetos e entender mais sobre as tecnologias envolvidas, navegue até a pasta correspondente e consulte a sua respectiva documentação (README).
